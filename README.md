# react-feedback-form

A simple web-based feedback form built using React and Bootstrap.  
This project allows users to submit feedback through email, give ratings on various topics, and add additional comments.

---

## 🛠 Tech Stack Used

- **React JS** – for building the user interface
- **React Bootstrap** – for form components and layout
- **Bootstrap 4** – for responsive styling
- **CSS** – for custom styles
